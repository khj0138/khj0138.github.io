# Render
## Render Component
* 물체를 그려주기 위해 존재하는 component
* mesh<br>
-> 물체의 형태에 대한 정보<br>
* material<br>
-> 물체를 그리는 데 있어서 지정된 형태에 대해 어떻게 그려질지 정의하는 정보<br>
* shared material<br>
-> default로 저장하여 사용하는 재질<br>
* dynamic material<br>
-> material 변경 시 임시 저장을 위한 재질 <br>
* current material<br>
-> 현재 사용 중인 재질 <br>
